Theme and Color paletes

I have decided to go with a playful cartoon style theme that has a hint of coding style to it. This is because not only are the colors use bright and visible, but it also gives the website a fun and playful feel to it that provides a sense of comfort to the user.

Above I show the color palette that I have chosen for the website. The colors are bright and visible, they are also easy on the eyes.

This is how the homepage is supposed to look like with the color scheme applied to it. This is the light mode of the website, I have also created a dark mode for the website that can be toggled by the user.

These wireframes also showcase the typefaces that I have chosen for the website. I have chosen to go with a DynaPuff typeface for the headings because it gives the website a cartoonish feel to it adding to the playful theme of the color scheme. The reason I only chose to use this font for the headings is because the headings are gonna have a fill color as illustrated, and this font looks good with a fill color but doesn't work so well on the body text. For this reason the body text will use the JetBrains Mono typeface, this typeface is a coding typeface and will contribute to the coding theme of the website.
